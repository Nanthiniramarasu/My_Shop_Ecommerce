## Description
MyShop is a full-stack e-commerce web application developed using Django.  
The application allows users to browse products, add items to the cart, place orders, and view order summaries.  
It also includes authentication features for user login and signup.

## Features
- User registration and login
- Product listing with details
- Add to cart functionality
- Cart item quantity update and removal
- Order placement and order summary
- Responsive and user-friendly UI

## Technologies Used
- Python
- Django
- HTML
- CSS
- Bootstrap
- JavaScript
- SQLite / MySQL
